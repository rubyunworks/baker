def show_help
  puts "\nbake v.09\n\n  USAGE: bake [options...] <command> [arguments...]\n\n  CATALOG COMMANDS:\n\n    sync\n        Syncronize local catalogs with masters.\n\n    find <arg>\n        Search the package catalog for all matching recipes.\n        Argument is a grep expression. (Currently only searches on file name.)\n\n    catd\n        Outputs the local path of the recipe catalog.\n\n    info <program>\n        Provides a description of the specified progam.\n\n    dep <programs...>\n        Displays a dependency tree.\n\n    con <programs...>\n        Displays confilcts.\n\n    new <type> <program> <version>\n        Creates a new receipe of given type. (Type is case sensitive.)\n\n    edit <program--version>\n        Edit the recipe with default editor.\n        (The editor is determined by enviornment variable 'EDITOR'.)\n\n  BUILD COMMANDS:\n\n    cast <programs...>\n        The cast processes does the following in order:\n        1) fetch 2) extract 3) patch 4) prepare 5) compile 6) install\n        The following commands can be used to do these individually.\n\n    fetch <programs...>\n        Fetches the source.\n\n    extract <programs...>\n        Extracts the source. (Will fetch if -b is set)\n\n    patch <programs...>  (NOT YET IMPLEMENTED)\n        Patches the source. (Will extract if -b is set)\n\n    prepare <programs...>\n        Prepares the source. (Will patch if -b is set)\n\n    compile <programs...>\n        Compiles the source. (Will prepare if -b is set)\n\n    install <programs...>\n        Installs the program. (Will compile if -b is set)\n        In batch mode this is the same as cast.\n\n    uninstall <programs...>\n        Uninstalls the program(s). This is not recommended if you did\n        not use this program to install.\n\n    purge <programs...>\n        Purges source. Completely removes the source. Ready for fresh fetch.\n\n  OTHER COMMANDS:\n\n    help\n        Displays this help information. (Same as -h or --help option)\n\n  OPTIONS:\n\n    -h  --help\n        Display this help information.\n\n    -b  --batch\n        Batch mode will back track operations to accomplish task.\n\n    -f  --force\n        Force downloads and extractions.\n\n    -i  --ignore\n        Ignore dependencies and conflicts.\n\n    -r  --redo\n        Will recompile and reinstall even if program is already installed.\n\n    -v  --verbose\n        Provides extra verbose processing information.\n\n    -p  --pretend  (NOT YET IMPLEMENTED)\n        Pretend, just to see what would happen.\n\n    -u  --use <file>\n        Use the specified configuration file for this build,\n        but does not set configuration.\n\n"
end
